# FirstSip

This is empty project.
Use it to explore iex and runtime.

```
iex.bat -S mix
```

Then you can try commands like:

```elixir
FirstSip.taste()
Application.loaded_applications()
Application.spec(:elixir)
:code.all_available()
:erlang.module_info(:exports)
Logger.module_info(:exports)
:logger.info("hello")
```
